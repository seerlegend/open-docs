  - AppID (通过合作沟通群获取)
  - AppSercet (通过合作沟通群获取)

  ```php
  $appid      = 2001234;

  $timestamp  = time(); // 1699943199 current timestamp
  ```

  ```bash
  curl --location 'https://dev-open.wowseer.com/api/report/birth/all' \
    --header 'Authorization: 2001234.1699943199.bf08195e265a83e13630531519708becdb66b602263b628edd672343a7b9b187' \

    --data '{

        "sex": 1,
        "solar": "2023-10-01",
        "times": "11:00-12:59"
    }'
  ```

## 接口说明
  - 请求方式 `POST`

  - 接口地址 `https://{{host}}/report/birth/{{type}}`

  - 路由参数type的值
    - `all`   全能报告
    - `love`  恋爱报告
    - `king`  人格报告 
    - `life`  命运报告
    - `relate`  人际报告 
    - `career`  职业报告 

  - header头参数lang的值
    - `sc` 简体中文
    - `tc` 繁体中文
    - `ko` 汉语

  - JSON参数
>
  |参数|必选|类型|说明|
  |:-----  |:-------:|:-----:|:-----  
  |name    |`ture`  |string|姓名                          |
  |sex     |`true`  |int   |性别 1=男，2=女
  |solar   |`true`  |string|生日 YYYY-MM-DD 类似2004-05-14 其中年份不得小于1900年
  |times   |false   |string|具体出生时辰 例如 准确时间12:00 或模糊时间 12:00-12:59
  |title   |false   |string|客户标题 `<50` 字(使用`data.url`预览时，html标题将显示此文字)
  |brand   |false   |string|客户品牌图片地址，如`https://image-home.com/branch.png` `<200` 字(使用`data.url`预览时，html底部将显示此图片)


  - 响应内容
> 
  |返回字段|字段类型|说明                              |
  |-----   |:------|:-----------------------------   |
  |code |int    |返回结果状态。`0` 正常；`>0` 错误。   |
  |msg  |string |提示信息                      |
  |data |object |        `{"url":"", "code":""}`               |
  |data.url |string |        通过HiSeer提供的H5链接预览报告内容               |
  |data.code |string |       报告码               |
